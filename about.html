tewUPs — Night Map

Why we made this

We created this night safety map to document security outposts and poorly-lit areas on campus so students can make safer route choices at night. It’s a collaborative site: if you notice a dark stretch or a new security post, tell the team and we’ll add it.

How to use the map

  1. Open the map (home). Markers show known security outposts and circles show dark areas.
  2. Contact the group if you want to suggest edits — for now, we update manually to verify each report.
